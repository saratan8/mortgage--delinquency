文档记录了每一个文件的内容和作用：
unemployment.zip：BLSapi的月度地区unemployment rate，内有00-09，10-19，20-23年各地区的unemployment rate的数据。
unemployment_data.csv：预处理格式后的unemployment rate，包含lag 3个月，lag 6个月
fm_hpi_master_file: fannie mae的月度地区hpi index，内有1975-2023年各地区的hpi的seasonally adj和non-adj数据。
hpi_data.csv：预处理格式后的hpi，包含lag 3个月，lag 6个月
***unemploy_hpi_merged.csv: 根据地区时间合并后的unemploy_hpi（含lag）的数据。
***layout：fannie mae的原文件的108col的标签
Sf-loan-performance-sample: fannie mae的sample file
***Performance_all.zip：fannie mae的原文件，2000-2023年4个季度
CR_mortgage.ipynb：Nov 22的code，未更新。
